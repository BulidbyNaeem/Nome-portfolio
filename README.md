##  🌐Naeem Khan-Portfolio Website:

A modern, responsive, and professional portfolio website built to showcase my frontend development skills, projects, and personal brand.
---

## 🚀 Live Demo:

👉 https://nome-khan-potfolio.vercel.app/

---
## 📌 Features

- ✨ Clean & modern UI design
- 📱 Fully responsive (mobile, tablet, desktop)
- 🎯 Project showcase section
- 🧑‍💻 About me section
- 📬 Contact form (Formspree integration)
- 🛠 Skills section with visual presentation
- 🔐 Admin panel for managing projects (local storage based)
- 🎨 Smooth animations and transitions

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Responsive Design
- Local Storage API
